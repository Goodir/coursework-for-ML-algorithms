# Курсовая работа по классическому машинному обучению

Тема работы: прогнозирование эффективности химических соединений для разработки лекарственных препаратов.

## Задачи работы

В рамках курсовой были решены следующие задачи:

* регрессия IC50
* регрессия CC50
* регрессия SI
* классификация IC50 выше медианного значения
* классификация CC50 выше медианного значения
* классификация SI выше медианного значения
* классификация SI выше 8

## Структура проекта

| Файл                                | Содержание                                          |
| ----------------------------------- | --------------------------------------------------- |
| `coursework_data.xlsx`              | исходный датасет                                    |
| `EDA.ipynb`                         | первичная очистка данных и исследовательский анализ |
| `regression_ic50.ipynb`             | построение моделей регрессии для IC50               |
| `regression_cc50.ipynb`             | построение моделей регрессии для CC50               |
| `regression_si.ipynb`               | построение моделей регрессии для SI                 |
| `classification_ic50.ipynb`         | классификация IC50 выше медианы                     |
| `classification_cc50.ipynb`         | классификация CC50 выше медианы                     |
| `classification_si.ipynb`           | классификация SI выше медианы                       |
| `classification_si8.ipynb`          | классификация SI выше 8                             |
| `coursework_whitepaper_report.pdf`  | итоговый аналитический отчет                        |

## Что было сделано

Сначала данные были очищены от пропусков, дублей, константных признаков и полных дублей признаков. После этого был проведен EDA: анализ распределений целевых переменных, проверка связи между IC50, CC50 и SI, анализ корреляций признаков с таргетами, проверка мультиколлинеарности и анализ групп дескрипторов.

Отдельно учитывалось, что SI является расчетным показателем и связан с IC50 и CC50. Поэтому IC50, CC50 и SI не использовались как признаки друг для друга, чтобы избежать утечки данных.

## Как запустить

Проект выполнен в формате Jupyter Notebook.

Основные библиотеки:

```bash
pandas
numpy
scikit-learn
matplotlib
plotly
openpyxl
```

Ноутбуки можно запускать последовательно:

1. `EDA.ipynb`
2. регрессионные ноутбуки
3. классификационные ноутбуки

Итоговые результаты и общий анализ собраны в файле `coursework_whitepaper_report.pdf`.
